| **. 인재상**                                        |
| 1. 가보지 않은 길을 두려워하지 않는 사람 **(도전 의식)**             |
| 2. 무슨 일이든 본질만 남기고 처음부터 다시 생각하는 사람 **(분석력, 차분함)** |

| 5. 세상을 선하게 바꾸려고 노력하는 사람 **(성실함, 선함)**            |
