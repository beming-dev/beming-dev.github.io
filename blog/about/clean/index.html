    • Clean Architecture with Nest.js

    March 28, 2025

    к·ңлӘЁ мһҲлҠ” мҶҢн”„нҠёмӣЁм–ҙ к°ңл°ңмқ„ н•ҳлӢӨліҙл©ҙ н•„м—°м ҒмңјлЎң лӢӨмқҢкіј к°ҷмқҖ л¬ём ңл“Өм—җ л§ҲмЈјн•©лӢҲлӢӨ.

    "мҳҒн–Ҙмқ„ л°ӣлҠ” кіімқҙ л„Ҳл¬ҙ л§Һм•„ мҲҳм •мқҙ нһҳл“ӨлӢӨ" "мқёмӣҗмқҙ лҠҳм–ҙлӮҳлҸ„ мғқмӮ°м„ұмқҙ к·ёлӢҘ лҶ’м•„м§Җм§Җ м•ҠлҠ”лӢӨ."

    Clean ArchitectureлҠ” мҪ”л“ң к°„ мқҳмЎҙм„ұмқ„ мӨ„м—¬ мҪ”л“ңмқҳ мң м§Җ ліҙмҲҳм„ұмқ„ лҶ’мқҙкі , ліҖкІҪмқ„ мң м—°н•ҳкІҢ н•©лӢҲлӢӨ.

    мөңмў…м ҒмңјлЎңлҠ” м•„мЈј м ҒмқҖ мқёл ҘмңјлЎңлҸ„ мӢңмҠӨн…ңмқ„ мң м§Җ, ліҙмҲҳ, нҷ•мһҘ н•  мҲҳ мһҲкІҢ лҗ©лӢҲлӢӨ.

    мқҙ нҸ¬мҠӨнҢ…мқҖ Clean Architecture мұ…мқ„ кё°л°ҳмңјлЎң, м ңк°Җ кІҪн—ҳн•ҙліё Clean Architectureм—җ кҙҖн•ң кёҖмһ…лӢҲлӢӨ. лӘЁл“  мҳҲмӢңлҠ” TypescriptмҷҖ Nest.js кё°л°ҳмһ…лӢҲлӢӨ.

    мқҳмЎҙ

    мҷң мҪ”л“ң к·ңлӘЁк°Җ м»Өм§Җл©ҙ м җм җ мҲҳм •, нҷ•мһҘмқҙ м–ҙл ӨмӣҢм§Ҳк№Ңмҡ”? мқҙлҘј мқҙн•ҙн•ҳкё° мң„н•ҙ лЁјм Җ мқҳмЎҙмқҙлқјлҠ” к°ңл…җмқ„ мқҙн•ҙн•ҙм•ј н•©лӢҲлӢӨ.

    Module

    мҡ°лҰ¬лҠ” мҪ”л“ңлҘј мһ‘м„ұн•  л•Ң, н•ҳлӮҳмқҳ нҢҢмқјм—җ лӘЁл“  мҪ”л“ңлҘј мһ‘м„ұн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. Javaмқҳ import, Javascriptмқҳ import, require, c, c++мқҳ include к°ҷмқҖ мҪ”л“ңлҘј нҶөн•ҙ мҷёл¶Җ лқјмқҙлёҢлҹ¬лҰ¬, нҳ№мқҖ лӢӨлҘё лӘЁл“Ҳмқҳ мҪ”л“ңлҘј л¶Ҳлҹ¬мҳӨкІҢ лҗ©лӢҲлӢӨ.

    л°”лЎң мқҙ л¶Җл¶„м—җм„ң мҪ”л“ңл“Ө к°„мқҳ мқҳмЎҙм„ұмқҙ мғқкё°кІҢ лҗ©лӢҲлӢӨ.

    мқјлЎҖлЎң, м–ҙл–Ө н”„лЎңм қнҠём—җ AлӘЁл“Ҳ, BлӘЁл“Ҳл§Ң мЎҙмһ¬н•ңлӢӨкі  к°Җм •н•ҙлҙ…мӢңлӢӨ. мқҙ л•Ң, AлӘЁл“Ҳм—җм„ңлҠ” BлӘЁл“Ҳмқ„ import н•ҙм„ң мӮ¬мҡ©н•ҳкі  мһҲмҠөлӢҲлӢӨ.

    л¶Ҳм•Ҳм •н•ң Module

    м•„л¬ҙлҸ„ AлӘЁл“Ҳмқ„ importн•ҳм§Җ м•Ҡм§Җл§Ң, AлӘЁл“ҲмқҖ BлӘЁл“Ҳмқҳ мҪ”л“ңлҘј мӮ¬мҡ©н•ҳкі  мһҲмҠөлӢҲлӢӨ. мқҙ л•Ң, A лӘЁл“Ҳмқҳ мҪ”л“ңлҘј ліҖкІҪн•ҳл©ҙ м–ҙл–Ёк№Ңмҡ”?

    AлӘЁл“Ҳм—җм„ң мҪ”л“ңлҘј м•„л¬ҙлҰ¬ мҲҳм •н•ҙлҸ„, м–ҙл–Ө мҪ”л“ңлҸ„ мҳҒн–Ҙмқ„ л°ӣм§Җ м•ҠмҠөлӢҲлӢӨ. мҰү, A лӘЁл“ҲмқҖ мҲҳм •м—җ мһҗмң лЎӯмҠөлӢҲлӢӨ. мҡ°лҰ¬лҠ” мқҙлҹ° мғҒнғңлҘј "л¶Ҳм•Ҳм •н•ҳлӢӨ" лқјкі  н•©лӢҲлӢӨ.

    м•Ҳм •м Ғмқё Module

    л°ҳл©ҙ, B лӘЁл“ҲмқҖ A лӘЁл“Ҳм—җм„ң нҳём¶ңлҗҳм§Җл§Ң, м•„л¬ҙ лӘЁл“ҲлҸ„ importн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. BлӘЁл“Ҳмқҳ мҪ”л“ңлҘј мҲҳм •н•ҳл©ҙ м–ҙл–»кІҢ лҗ к№Ңмҡ”? BлӘЁл“Ҳмқҳ мҲҳм •м—җ л”°лқј, A лӘЁл“Ҳк№Ңм§Җ мҲҳм •н•ҙм•ј н•  кІҪмҡ°к°Җ мғқкёё кІғмһ…лӢҲлӢӨ.

    мҰү, BлӘЁл“ҲмқҖ мҲҳм •мқҙ мһҗмң лЎӯм§Җ м•ҠмҠөлӢҲлӢӨ. мҡ°лҰ¬лҠ” мқҙлҹ° мғҒнғңлҘј "м•Ҳм •м ҒмқҙлӢӨ" лқјкі  н•©лӢҲлӢӨ.

    к·ёл ҮлӢӨл©ҙ лӘЁл“  лӘЁл“Ҳмқҙ м•Ҳм •м Ғмқҙл©ҙ мўӢмқ„к№Ңмҡ”? к·ёлҹј мҪ”л“ңлҘј мҲҳм •н•  мҲҳ м—ҶмңјлҜҖлЎң мўӢм§Җ м•ҠмҠөлӢҲлӢӨ. к·ёлҹј лӘЁл“  лӘЁл“Ҳмқҙ л¶Ҳм•Ҳм •н•ҳл©ҙ мўӢмқ„к№Ңмҡ”? мўӢкІ м§Җл§Ң мқҙлҠ” л¶Ҳк°ҖлҠҘн•©лӢҲлӢӨ.

    м–ҙл–Ө лӘЁл“Ҳмқҙ л¶Ҳм•Ҳм •н•ҙм§Җл©ҙ м–ҙл–Ө лӘЁл“ҲмқҖ м•Ҳм •м ҒмқҙкІҢ лҗ©лӢҲлӢӨ. л”°лқјм„ң мҡ°лҰ¬лҠ” мҪ”л“ңлҘј л¶Ҳм•Ҳм •н•ң лӘЁл“Ҳкіј м•Ҳм •н•ң лӘЁл“ҲлЎң л¶„лҰ¬н•ҙм•ј н•©лӢҲлӢӨ.

    м¶”мғҒнҷ”мҷҖ кө¬мІҙнҷ”

    мһ мӢң мқҙм•јкё°лҘј л„ҳм–ҙмҷҖ лҙ…мӢңлӢӨ.

    лӢӨмқҢкіј к°ҷмқҖ мҪ”л“ңлҘј ліё м Ғ мһҲмқ„кІҒлӢҲлӢӨ.

    interface Logger {
  log(message: string): void;
}

    class ConsoleLogger implements Logger {
  log(message: string): void {
    console.log(`[Console] ${message}`);
  }
}

    interfaceлҘј л§Ңл“Өкі , к·ё interfaceм—җ л§һлҠ” classлҘј кө¬нҳ„ н–ҲмҠөлӢҲлӢӨ.

    мқҙ мҪ”л“ңм—җм„ң userProcessorмқҖ м¶”мғҒнҷ” л¶Җл¶„, AdminUserProcessorмқҖ кө¬мІҙнҷ” л¶Җл¶„мқҙлқј л¶ҖлҰ…лӢҲлӢӨ.

    мҷң көімқҙ мҪ”л“ңлҘј мқҙл ҮкІҢ лӮҳлҲҢк№Ңмҡ”? к·ёкұҙ л°”лЎң мқён„°нҺҳмқҙмҠӨ(м¶”мғҒнҷ” л¶Җл¶„)мқҖ м•Ҳм •м Ғмқҙкі , кө¬нҳ„мІҙ л¶Җл¶„мқҖ л¶Ҳм•Ҳм •н•ҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ.

    м¶”мғҒнҷ”лҘј нҶөн•ң мқҳмЎҙм„ұ м ңм–ҙ

    мқҙм ң л‘җ к°Җм§Җ к°ңл…җмқ„ мқ‘мҡ©н•ҙлҙ…мӢңлӢӨ.

    м•„к№ҢмҷҖ к°ҷмқҙ AлӘЁл“Ҳм—җм„ң BлӘЁл“Ҳмқ„ нҳём¶ңн•ҳл Ө н•©лӢҲлӢӨ. к·ёлҹ°лҚ° л§Ңм•Ҫ, BлӘЁл“Ҳмқҙ м¶”мғҒнҷ”лҗң мҪ”л“ңлҘј к°Җм§Җкі  мһҲлӢӨкі  мғқк°Ғн•ҙлҙ…мӢңлӢӨ.

    AлӘЁл“Ҳм—җм„ң BлӘЁл“Ҳмқҳ м¶”мғҒнҷ”лҘј importн•ҳм—¬ мӮ¬мҡ©н•ҳл©ҙ м–ҙл–Ёк№Ңмҡ”? м¶”мғҒнҷ”лҠ” м•Ҳм •м Ғмқҙкё° л•Ңл¬ём—җ мқҙлҠ” л§Өмҡ° мһҗм—°мҠӨлҹҪмҠөлӢҲлӢӨ. м¶”мғҒнҷ”лҗң мҪ”л“ңлҠ” ліҖкІҪмқҙ кұ°мқҳ м—Ҷкё° л•Ңл¬ём—җ AлӘЁл“Ҳмқҙ мҲҳм •м—җ мҳҒн–Ҙмқ„ л°ӣмқ„ мқјмқҙ мһҳ м—Ҷмқ„ кІғмһ…лӢҲлӢӨ.

    мқҙлҹ° мӢқмңјлЎң мҪ”л“ңмқҳ лӘЁл“  мқҳмЎҙмқ„ м¶”мғҒнҷ”м—җ н•ҳл©ҙ м–ҙл–Ёк№Ңмҡ”? мҡ°лҰ¬лҠ” кө¬нҳ„мІҙлҘј мһҗмң лЎӯкІҢ мҲҳм •н•ҳл©ҙм„ң мҪ”л”©н•  мҲҳ мһҲмқ„ кІғмһ…лӢҲлӢӨ.

    мқҙ к°ңл…җмқҙ л°”лЎң "мқҳмЎҙм„ұ м ңм–ҙ"мһ…лӢҲлӢӨ. мң„мқҳ мқҙлҜём§ҖмҷҖ к°ҷмқҙ мқҳмЎҙм„ұмқҳ л°©н–Ҙмқ„ л°”кҫём–ҙ м„Өкі„н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    мқҙм ң м•Ҳм •м Ғмқё л¶Җл¶„мқ„ м¶”мғҒнҷ”лЎң, л¶Ҳм•Ҳм •н•ң л¶Җл¶„мқ„ кө¬нҳ„мІҙлЎң мҪ”л“ңлҘј лӮҳлҲ„лҠ” кІғм—җ м„ұкіөн–ҲмҠөлӢҲлӢӨ.

    мқҙлҠ” л§Өмҡ° мқҙмғҒм Ғмқҙм§Җл§Ң н•ңк°Җм§Җ н•Ём •мқҙ мЎҙмһ¬н•©лӢҲлӢӨ. м¶”мғҒнҷ”м—җ мқҳмЎҙн•ңлӢӨл©ҙ, кө¬нҳ„мІҙ мҪ”л“ңлҠ” лӘЁлҘҙлҠ”кІғ м•„лӢҢк°Җмҡ”? мӢӨм ң мҪ”л“ңмқҳ лҸҷмһ‘мқҖ кө¬нҳ„мІҙк°Җ н• н…җлҚ°, м¶”мғҒнҷ”лҘј importн•ҙм„ң мӮ¬мҡ©н•ҳл©ҙ мҪ”л“ңк°Җ лҸҷмһ‘н•ҳм§Ҳ м•ҠмҠөлӢҲлӢӨ.

    мқҳмЎҙм„ұ мЈјмһ…

    к·ёлһҳм„ң н•„мҡ”н•ң к°ңл…җмқҙ л°”лЎң Dependency Injection(DI), мқҳмЎҙм„ұ мЈјмһ…мһ…лӢҲлӢӨ. м¶”мғҒнҷ”м—җ мқҳмЎҙн•ҳлҠ” лӘЁл“Ҳл“Өм—җкІҢ, кө¬нҳ„мІҙмқҳ лӮҙмҡ©мқ„ м•Ңл ӨмЈјлҠ” кІғмқҙмЈ .

    к·ёлҹј м–ҙл–Ө л°©лІ•мңјлЎң мқҳмЎҙм„ұмқ„ мЈјмһ…н•  мҲҳ мһҲмқ„к№Ңмҡ”?

    мң„мқҳ мҳҲмӢңлҘј лӢӨмӢң к°Җм ёмҷҖм„ң м•Ңм•„лҙ…мӢңлӢӨ.

    interface Logger {
  log(message: string): void;
}

    class ConsoleLogger implements Logger {
  log(message: string): void {
    console.log(`[Console] ${message}`);
  }
}

    мғқм„ұмһҗ мЈјмһ…

    нҒҙлһҳмҠӨк°Җ мғқм„ұлҗ  л•Ң н•„мҡ”н•ң мқҳмЎҙм„ұмқ„ м „лӢ¬ л°ӣмҠөлӢҲлӢӨ.

    class UserService {
  constructor(private logger: Logger) {}

  registerUser(username: string) {
    this.logger.log(`мң м Җ л“ұлЎқ мҷ„лЈҢ: ${username}`);
  }
}

// мӮ¬мҡ© мҳҲмӢң
const logger = new ConsoleLogger();
const userService = new UserService(logger);
userService.registerUser('нҷҚкёёлҸҷ');

    мҪ”л“ңлҘј ліҙл©ҙ, нҒҙлһҳмҠӨм—җм„ңлҠ” Logger мқён„°нҺҳмқҙмҠӨл§Ң мӮ¬мҡ©н•ҳм§Җл§Ң мҷёл¶Җм—җм„ң мғқм„ұн•  л•Ң Loggerмқҳ лӮҙмҡ©мқ„ мЈјмһ…н•ҙмӨҚлӢҲлӢӨ.

    л©”м„ңл“ң мЈјмһ…

    л©”м„ңл“ңлҘј нҳём¶ңн•  л•Ңл§ҲлӢӨ мқҳмЎҙм„ұмқ„ м „лӢ¬н•ҙмӨҚлӢҲлӢӨ. л§ӨлІҲ лӢӨлҘё мқҳмЎҙм„ұмқ„ мң м—°н•ҳкІҢ м „лӢ¬н•  мҲҳ мһҲм§Җл§Ң, мҪ”л“ңк°Җ ліөмһЎн•ҙм§‘лӢҲлӢӨ.

    class UserService {
  registerUser(username: string, logger: Logger) {
    // мң м Җ л“ұлЎқ лЎңм§Ғ
    logger.log(`мң м Җ л“ұлЎқ мҷ„лЈҢ: ${username}`);
  }
}

// мӮ¬мҡ© мҳҲмӢң
const userService = new UserService();
const logger = new ConsoleLogger();

userService.registerUser('нҷҚкёёлҸҷ', logger);

    Setter мЈјмһ…

    мқҳмЎҙм„ұмқҙ н•„мҡ”н•ң кіім—җ SetterлҘј л§Ңл“Өм–ҙ лӮҳмӨ‘м—җ м„ нғқм ҒмңјлЎң мЈјмһ…н•©лӢҲлӢӨ.

    class UserService {
  private logger?: Logger;

  setLogger(logger: Logger) {
    this.logger = logger;
  }

  registerUser(username: string) {
    if (this.logger) {
      this.logger.log(`мң м Җ л“ұлЎқ мҷ„лЈҢ: ${username}`);
    } else {
      console.log('Loggerк°Җ м—ҶмҠөлӢҲлӢӨ.');
    }
  }
}

// мӮ¬мҡ© мҳҲмӢң
const userService = new UserService();
userService.setLogger(new FileLogger());
userService.registerUser('нҷҚкёёлҸҷ');

    м–ҙл””м„ң мқҳмЎҙм„ұмқ„ мЈјмһ…н• к№Ң?

    мқҳмЎҙм„ұмқ„ мЈјмһ…н•ҳлҠ” лӢӨм–‘н•ң л°©лІ•мқ„ м•Ңм•ҳмҠөлӢҲлӢӨ. к·ёлҹ¬лӮҳ н•ңк°Җм§Җ лҳҗ кі лҜјмқҙ мғқк№ҒлӢҲлӢӨ. м–ҙл””м„ң мқҳмЎҙм„ұмқ„ мЈјмһ…н•ҙмӨҳм•ј н• к№Ңмҡ”? мқҳмЎҙм„ұмқ„ мЈјмһ…н•ҙмЈјкё° мң„н•ҙм„  кө¬нҳ„мІҙ мҪ”л“ңлҘј нҳём¶ңн•ҙм•ј н•ҳлҠ”лҚ°, м–ҙл–»кІҢ ліҙл©ҙ кө¬нҳ„мІҙм—җ мқҳмЎҙн•ҳкІҢ лҗңлӢӨкі  ліј мҲҳ мһҲмҠөлӢҲлӢӨ.

    к·ёлһҳм„ң ліҙнҶө мқҳмЎҙм„ұ мЈјмһ…мқҖ "main" м»ҙнҸ¬л„ҢнҠём—җм„ң м „л¶Җ мІҳлҰ¬н•©лӢҲлӢӨ. мқҙлҰ„мқҙ mainмқҙ м•„лӢҲм–ҙлҸ„, н”„лЎңк·ёлһЁмқҳ entry pointк°Җ к·ё м—ӯн• мқ„ н•ҳкІҢ лҗ©лӢҲлӢӨ.

    мқҙл ҮкІҢ н•ҳл©ҙ м–»лҠ” мқҙм җмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

    • н•ң кіім—җм„ң к°қмІҙлҘј мғқм„ұн•ҳм—¬ мӨ‘м•ҷ м§‘мӨ‘нҷ” н•ңлӢӨ.
    • мқҳмЎҙм„ұмқҳ нқҗлҰ„мқҙ лӘ…нҷ•н•ҙм§„лӢӨ.
    • кө¬нҳ„мІҙм—җ мқҳмЎҙн•ҳлҠ” кІғмқ„ mainмқҙ нҳјмһҗ л§Ўмқ„ мҲҳ мһҲлӢӨ.
